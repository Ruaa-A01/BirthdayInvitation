# Birthday Invitation Card ♥︎

A small, aesthetic birthday invitation web page built with HTML, CSS and a subtle JavaScript interaction.

---

## Overview

This project features a responsive digital birthday invitation designed with simplicity and visual charm in mind.

- A Responsive Invitation Card (`index.html`) Styled Using `style.css`.
- Decorative Images Are Stored In `images/` Folder & Displayed In A Grid Layout `decor-grid` Gallery.
- A Lightweight `scripts.js` Adds A Confetti Burst Effect When The Card Is Clicked.

<img src="./screenshots/birthdayinvite.png" alt="Birthday Invite" width="800"/>

---

## Project Structure

- `index.html` — Main Invitation Page (Open In A Browser)
- `style.css` — Visual Theme, Layout & Styling
- `scripts.js` — Handles The Click-Activated Confetti Animation
- `images/` — Local Decorative Assets Used Throughout The Page

---

## How To View

1. Open `index.html` In Your Browser.
2. Click On The Invitation Card To Activate The Confetti Effect.
